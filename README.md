# panel-pg-pages

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能用于展示信息、提供工具界面或作为静态资源访问。仓库本身不针对特定域名或网站，而是作为一个通用的页面集合存储与发布空间。

## 目录说明

```
panel-pg-pages/
├── pages/          # 存放所有独立 HTML 页面文件
├── assets/         # 页面引用的静态资源（如图片、样式表、脚本）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件对应一个独立页面，文件名即页面标识。
- `assets/`：按需存放页面所需的公共资源，建议按类型或页面归类。

## 页面归档说明

- 每个页面均为独立 HTML 文件，可直接在浏览器中打开或通过静态服务器访问。
- 页面之间无依赖关系，可单独更新或删除。
- 归档的页面可能随时间更新或移除，不保证向后兼容。
- 若需引用某个页面，建议直接链接到该文件的原始地址。

## 维护说明

- 新增页面：在 `pages/` 目录下添加 HTML 文件，并在 `assets/` 中补充所需资源。
- 更新页面：直接修改对应 HTML 文件，注意保持资源路径正确。
- 删除页面：移除对应 HTML 文件及不再使用的资源文件。
- 提交前请确保页面功能正常，无外部依赖缺失。

## 许可

本仓库内容仅供归档与发布使用，具体页面的权利归属请参见各文件内的声明。
